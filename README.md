# 🧠 BrainDumper

**BrainDumper** — это интеллектуальный Telegram-бот для быстрого захвата и организации ваших мыслей, идей и задач. Просто отправьте сообщение боту, а он самостоятельно классифицирует его и сохранит в удобном формате, помогая не потерять ни одной важной мысли и контролировать дедлайны.

## 🌟 Возможности

*   **📥 Быстрый захват мыслей:** Отправьте сообщение в Telegram, и бот автоматически определит его тип. 
*   **🏷️ Умная классификация:** Контент автоматически распределяется по трём категориям:
    *   **💡 Идеи:** Для творческих концепций и новых проектов. Планируется создание отдельных блокнотов и инструментов для их развития.
    *   **💭 Мысли:** Для быстрых заметок, наблюдений и всего, что нужно просто сохранить, не превращая в задачу.
    *   **✅ Задачи:** Для дел, требующих выполнения. С поддержкой дедлайнов и напоминаний.
*   **⏰ Напоминания о задачах:** Бот не даст забыть о важных сроках, присылая уведомления по заданному дедлайну.(Планируется трекер задач для полного цикла управления делами).
*   **🗂️ Организация идей:** (В разработке) Специальные "блокноты" для группировки идей по темам и инструменты для их проработки и развития.
*   **📦 Хранение заметок:** Все ваши записи надёжно сохраняются и структурируются для лёгкого поиска в будущем. 

## 🚀 Технологии
*   Язык: **Python 3.12**
*   Библиотека для Telegram API: **aiogram**
*   База данных: **выбираю**

## ⚙️ Установка и запуск

### Предварительные требования
*   Установленный Python 3.12
*   Git
*   Токен бота Telegram (получить у [@BotFather](https://t.me/BotFather))

### Инструкция по установке

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ваш_username/BrainDumper.git
    cd BrainDumper
    ```
2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
  source venv/bin/activate  # для Linux/Mac
  # venv\Scripts\activate  # для Windows
  ```
3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```
4. **Настройка переменных окружения:**
Создайте файл .env в корневой папке проекта и добавьте туда ваш токен:
```.env
BOT_TOKEN=ваш_токен_бота
DATABASE_URL=ссылка_на_базу_данных
PROXY_URL = адрес прокси для телеграмма
```
5. **Запустите бота**
   ```bash
   python main.py  # или bot.py, в зависимости от вашего главного файла
   ```
## 🎯 Использование

1.  Найдите своего бота в Telegram и начните диалог командой `/start`.
2.  Просто отправляйте боту любые текстовые сообщения.
